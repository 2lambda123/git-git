apply.ignoreWhitespace::
	When set to 'change', tells 'but apply' to ignore changes in
	whitespace, in the same way as the `--ignore-space-change`
	option.
	When set to one of: no, none, never, false tells 'but apply' to
	respect all whitespace differences.
	See linkbut:but-apply[1].

apply.whitespace::
	Tells 'but apply' how to handle whitespaces, in the same way
	as the `--whitespace` option. See linkbut:but-apply[1].
