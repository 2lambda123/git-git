but-citool(1)
=============

NAME
----
but-citool - Graphical alternative to but-cummit

SYNOPSIS
--------
[verse]
'but citool'

DESCRIPTION
-----------
A Tcl/Tk based graphical interface to review modified files, stage
them into the index, enter a cummit message and record the new
cummit onto the current branch.  This interface is an alternative
to the less interactive 'but cummit' program.

'but citool' is actually a standard alias for `but gui citool`.
See linkbut:but-gui[1] for more details.

BUT
---
Part of the linkbut:but[1] suite
