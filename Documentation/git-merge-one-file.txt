but-merge-one-file(1)
=====================

NAME
----
but-merge-one-file - The standard helper program to use with but-merge-index


SYNOPSIS
--------
[verse]
'but merge-one-file'

DESCRIPTION
-----------
This is the standard helper program to use with 'but merge-index'
to resolve a merge after the trivial merge done with 'but read-tree -m'.
