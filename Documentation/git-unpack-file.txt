but-unpack-file(1)
==================

NAME
----
but-unpack-file - Creates a temporary file with a blob's contents



SYNOPSIS
--------
[verse]
'but unpack-file' <blob>

DESCRIPTION
-----------
Creates a file holding the contents of the blob specified by sha1. It
returns the name of the temporary file in the following format:
	.merge_file_XXXXX

OPTIONS
-------
<blob>::
	Must be a blob id

BUT
---
Part of the linkbut:but[1] suite
