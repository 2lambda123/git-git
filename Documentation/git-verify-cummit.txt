git-verify-cummit(1)
====================

NAME
----
git-verify-cummit - Check the GPG signature of cummits

SYNOPSIS
--------
[verse]
'git verify-cummit' <cummit>...

DESCRIPTION
-----------
Validates the GPG signature created by 'git cummit -S'.

OPTIONS
-------
--raw::
	Print the raw gpg status output to standard error instead of the normal
	human-readable output.

-v::
--verbose::
	Print the contents of the cummit object before validating it.

<cummit>...::
	SHA-1 identifiers of Git cummit objects.
