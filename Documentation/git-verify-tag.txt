but-verify-tag(1)
=================

NAME
----
but-verify-tag - Check the GPG signature of tags

SYNOPSIS
--------
[verse]
'but verify-tag' [--format=<format>] <tag>...

DESCRIPTION
-----------
Validates the gpg signature created by 'but tag'.

OPTIONS
-------
--raw::
	Print the raw gpg status output to standard error instead of the normal
	human-readable output.

-v::
--verbose::
	Print the contents of the tag object before validating it.

<tag>...::
	SHA-1 identifiers of Git tag objects.
