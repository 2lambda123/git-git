but-version(1)
==============

NAME
----
but-version - Display version information about Git

SYNOPSIS
--------
[verse]
'but version' [--build-options]

DESCRIPTION
-----------
With no options given, the version of 'but' is printed on the standard output.

Note that `but --version` is identical to `but version` because the
former is internally converted into the latter.

OPTIONS
-------
--build-options::
	Include additional information about how but was built for diagnostic
	purposes.

GIT
---
Part of the linkbut:but[1] suite
